Buttons

The following are the button styles acceptable for the development of this site. The basic style is flat and no border.

Style 1 - base style

border: none; background-color: #C7CCDB; color: white; padding: 10px;

Style 2 - Hover effect

border: 1px solid #2A324B; background-color: #C7CCDB; color: white; padding: 10px;

Style 3 - basic transition

border: 1px solid #2A324B; background-color: #C7CCDB; color: white; padding: 10px; cursor: pointer;
